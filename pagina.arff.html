O Weka possui um formato próprio, o ARFF, o qual temos que descrever o domínio do atributo, pois o mesmo não pode ser obtido automaticamente pelo seu valor. Antes de aplicar os dados a qualquer algoritmo do pacote Weka estes devem ser convertidos para o formato ARFF que consiste basicamente de duas partes:

A primeira contém uma lista de todos os atributos, onde devemos definir o tipo do atributo ou os valores que ele pode representar. Quando utilizamos os valores estes devem estar entre “{ }“ separados por vírgulas.

A segunda parte consiste das instâncias, ou seja, os registros a serem minerados com o valor dos atributos para cada instancia separado por vírgula. A ausência de um item em um registro deve ser atribuída pelo símbolo “?”.

Para a criação de um arquivo .arff é só seguir os seguintes passos:

Primero Passo:

Primeiramente, para criar um arquivo ARFF, abra um editor de texto limpo, geralmente já vem instalado no seu sistema operacional, como o bloco de notas do Windows.

Segundo Passo:

Use a tag @Relation para definir o nome do arquivo (exemplo: @ relation Acampamento). Então para definir o nome do atributo use a tag @Attribute.

Terceiro Passo:

Para criar elementos com valores dos atributos pré-definidos utilize a tag @Data na ordem de criação dos atributos. Então clique em salvar como e salve o arquivo como .ARFF, sendo assim, o Weka poderá interpretar seu arquivo, conseguindo abri-lo e lendo seus valores.

Para abrir o arquivo .arff no Windows é só seguir os passos abaixo:

Primero Passo:

Clique com o botão direito do mouse no arquivo ARFF e escolha a opção Abrir com

Segundo Passo:

Em seguida, selecione a opção Escolher outro aplicativo e, usando Mais aplicativos abra a lista de aplicativos disponíveis

Terceiro Passo:

Por fim, selecione Procure outro aplicativo neste PC , aponte para a pasta em que Weka está instalado, marque a caixa Sempre usar este aplicativo para abrir os arquivos ARFF e confirme sua seleção clicando no botão OK

Para abrir o arquivo .arff no Mac Os é só seguir os passos abaixo:

Primero Passo:

Ao clicar com o botão direito do mouse no arquivo ARFF selecionado, abra o menu e escolha Informações

Segundo Passo:

Abra a seção Abrir com clicando em seu nome. Então selecione o software apropriado e salve suas configurações clicando em Alterar tudo

Terceiro Passo:

Por fim, a mensagem Esta alteração será aplicada a todos os arquivos com extensão ARFF deve aparecer. Clique no botão Continuar para confirmar sua escolha.
