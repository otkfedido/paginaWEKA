Historia Weka

Weka é uma coleção de algoritmos de aprendizado de máquina para tarefas de mineração de dados. Ele contém ferramentas para preparação de dados, classificação, regressão, agrupamento, mineração de regras de associação e visualização.

Para que usa-lo?

O cientista de dados precisa ser produtivo devido a diversas tarefas que lhe são atribuídas. Mas como ser produtivo com diversas bases para analisar, processar, e testar soluções? Por exemplo… Você precisa testar diferentes algoritmos de machine learning para saber qual funcionará melhor para o seu sistema? Ou ainda, você quer testar algumas técnicas de pré-processamento de dados e ver como seus modelos irão se comportar? Você tem um banco de dados relacional na sua empresa com informações valiosas mas não sabe como pode trabalhar com esses dados? Com o Weka é possível fazer todas essas tarefas com poucos cliques! O Weka simplifica muito o trabalho por ser uma ferramenta simples e intuitiva.

Como baixa-lo

Existem duas versões do Weka: Weka 3.8 é a versão estável mais recente e Weka 3.9 é a versão de desenvolvimento. Os novos lançamentos dessas duas versões são normalmente feitos uma ou duas vezes por ano. Para a borda sangrenta, também é possível fazer o download de instantâneos noturnos dessas duas versões. A versão estável recebe apenas correções de bugs e atualizações de recursos que não quebram a compatibilidade com seus lançamentos anteriores, enquanto a versão de desenvolvimento pode receber novos recursos que quebram a compatibilidade com seus lançamentos anteriores. O Weka 3.8 e 3.9 apresentam um sistema de gerenciamento de pacotes que facilita para a comunidade Weka adicionar novas funcionalidades ao Weka. O sistema de gerenciamento de pacotes requer uma conexão com a Internet para baixar e instalar pacotes. Instantâneos Todas as noites, uma captura instantânea do repositório Subversion com o código fonte Weka é obtida, compilada e montada em arquivos ZIP. Isso acontece para o ramo de desenvolvimento do software e o ramo estável. Aqueles que desejam as correções mais recentes antes do próximo lançamento oficial podem fazer o download desses instantâneos . Versão estável Weka 3.8 é a versão estável mais recente do Weka. Esse ramo da Weka recebe apenas correções e atualizações que não quebram a compatibilidade com versões anteriores do 3.8, embora os principais recursos novos possam estar disponíveis nos pacotes. Existem diferentes opções para baixar e instalar no seu sistema: janelas Clique aqui para baixar um arquivo executável de extração automática para Windows de 64 bits que inclui o OpenJDK Java VM 11 de 64 bits da Azul (weka-3-8-4-azul-zulu-windows.exe; 118 MB) Este executável instalará o Weka no seu menu de programas. O lançamento através do Menu do Programa ou atalhos usará automaticamente a JVM incluída para executar o Weka. Mac OS Clique aqui para baixar uma imagem de disco para Mac OS que contém um aplicativo Mac, incluindo o OpenJDK Java VM 11 de 64 bits da Azul (weka-3-8-4-azul-zulu-osx.dmg; 144 MB) Linux Clique aqui para baixar um arquivo zip para Linux que inclui o OpenJDK Java VM 11 da Azul de 64 bits (weka-3-8-4-azul-zulu-linux.zip; 129 MB) Primeiro descompacte o arquivo zip. Isso criará um novo diretório chamado weka-3-8-4. Para executar o Weka, mude para esse diretório e digite ./weka.sh Outras plataformas Clique aqui para baixar um arquivo zip contendo Weka (weka-3-8-4.zip; 53 MB) Primeiro descompacte o arquivo zip. Isso criará um novo diretório chamado weka-3-8-4. Para executar o Weka, mude para esse diretório e digite java -jar weka.jar Observe que o Java precisa estar instalado no seu sistema para que isso funcione. Observe também que o uso -jarsubstituirá sua variável CLASSPATH atual e usará apenas o weka.jar. Aplicações do dispositivo

Ao executar a ferramenta, tela que se abre é essa abaixo. Perceba que contém várias aplicações (Explorer, Experimenter, KnowledgeFlow, Workbench e Simple CLI) Explorer Esta aplicação é a mais usada, com esta é possível trabalhar de forma interativa. È onde iremos explorar as funções da ferramenta. Vamos focar nessa ferramenta, pois, nas outras podemos fazer muitas funções que contém nessa, mas de forma automatizada. Experimenter: Esta aplicação é usada para automatização de processos, é usada por exemplo, para rodar vários algoritmos de aprendizado de máquina de forma automática e visualizar todos os resultados de uma só vez. KnowledgeFlow: Esta aplicação permite o usuário criar fluxos de processos através de uma interface baseada em fluxo de dados. È possível arrastar componentes como filtros ou classificadores e conecta-los, criando assim um fluxo de processamento. Workbench: Esta aplicação combina todas as outras aplicações mostradas em uma unica interface. Essa pode ser interessante caso você tem que alternar entre a aplicação Explorer e Experimenter por muitas vezes. Simple CLI: Essa é uma interface de linha de comando que permite criar scripts shell para usar a API do Weka. Com esta API é possível rodar experimentos, algoritmos de machine learning e diversas outras tarefas. A vantagem é que você pode automatizar tarefas e integrar na sua aplicação.
